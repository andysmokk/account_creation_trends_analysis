# Account creation trends analysis

Collection of data to help analyze trends in account creation, user activity related to emails (sending, opens, clicks), and to evaluate behavior across categories such as sending intervals, account verification, and subscription status. The data will enable comparison of activity across countries, identification of key markets, and user segmentation based on various parameters
